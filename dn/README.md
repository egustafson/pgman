Design Notes
============

This folder contains design notes for this project.  Among the notes there are two types of files that are often used w/ AI code generation:
* `promptXXX.md` are prompt documents, often written by a human, that provide instructions and context for an AI agent to perform a task.
* `planXXX.md` are plan documents, often written by an AI agent, that outline the steps to accomplish a task.

Other documents, both human and AI generated may be found in this folder.
